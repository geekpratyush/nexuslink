# GraphQL Client

Query any GraphQL HTTP endpoint.

## How to use
1. Enter the **endpoint** URL.
2. Write a **query** (and optional **variables** JSON).
3. **Run** (`Ctrl+Enter`) — the pretty-printed JSON response appears on the right.

## Introspection
Click **Introspect** to fetch the schema (types + root query/mutation/subscription) using the standard introspection query — handy for discovering what a server exposes.

> Tip: try the *Countries GraphQL* or *Rick & Morty GraphQL* samples under **Samples (public)**.

WebSocket subscriptions are on the roadmap.
