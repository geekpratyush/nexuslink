# LDAP / Active Directory

> **Status:** on the roadmap (Phase 8.4).

The planned LDAP client (UnboundID SDK) will support:

- Connect with StartTLS / LDAPS, simple or SASL bind.
- Browse the directory information tree (DIT) and edit entries (LDIF view).
- A search dialog with a custom filter builder and predefined filters.

For directory-style browsing today, the **MongoDB**, **SQL**, and **object-storage** explorers use the same tree + details pattern LDAP will adopt.
